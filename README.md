# 🌤️ SkyPulse

SkyPulse is a full-stack web application that shows **Weather Forecast and Air Quality Index (AQI)** for any city.
Users can search cities, check current weather conditions, and view air quality levels in a simple and clean interface.

## 🚀 Features

* 🔎 City search with suggestions
* 📍 Detect weather using current location
* 🌡️ Current weather + hourly forecast
* 📅 7-day weather forecast
* 🌫️ Air Quality Index (AQI) display
* ⚠️ Error handling for invalid cities and API issues
* 🎨 Responsive and clean UI

## 🛠️ Tech Stack

Frontend: **React, JavaScript, HTML, CSS**
Backend: **Node.js, Express**

## 🌐 APIs Used

* Open-Meteo (Weather data)
* WAQI API (Air Quality data)

## ⚙️ Run Locally

Clone the repository

```bash
git clone https://github.com/your-username/skypulse.git
```

Install dependencies

```bash
npm install
```

Start frontend

```bash
npm run dev
```

Start backend

```bash
nodemon server.js
```
